# Faetropolis Website

A basic one-page product website showcasing Faetropolis, a Unity mobile AR game about fairies.

## Features
- Parallax scrolling effects
- Responsive design
- Sections: Hero, About, Features, Contact

## How to View
Open `index.html` in a web browser.
